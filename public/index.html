.ready()

  • A page can't be manipulated until the Document Object Model (DOM) is "ready" for JS code to execute.
  • $(function() {}) is the shorthand for $(document).ready(function() {})
  • vs. $(window).load(function() { ... }) (fires an event only after the page is fully loaded, including all frames, objects, and images)

Parent Div (with event)
Child Div

mouseout: 0 times

.hover(handlerIn, handlerOut)

$( selector ).hover( handlerIn, handlerOut ) is the shorthand for $( selector ).mouseenter( handlerIn ).mouseleave( handlerOut ).

Note: If you only provide one handler, .hover() will only run a .mouseenter() event.
